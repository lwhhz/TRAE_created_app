这是一个纯粹由TRAE Agent制作的基于flutter，符合MD3设计语言（但不美观）的简易天气软件。  
天气源使用高德地图API，一句使用了开发者[@Xygeng](https://github.com/xygengcn)提供的API，十分感谢。

```
//编译安卓apk
flutter build apk

//编译Windows应用程序
flutter build windows
```
Windows 安装时会出现证书问题，这个问问ai就知道怎么解决了很简单😊
